Conhecendo a Estrutura HTML5
DOCTYPE HTML: nessa instrução informamos para o browser qual a especificação do código, esta declaração deve ser definida antes da tag HTML. HTML lang="pt-br": Além de definimos que este e o começo do nosso html, nesta parte também informamos para o Browser que o idioma do nosso arquivo será pt-br head/head: nesta área definimos algumas configurações onde podemos destacar o titulo e também os meta dados que são um conjunto de informações sobre o documento e do conteúdo publicado. title/title: nesta tag definimos o titulo da pagina, no exemplo acima definimos como Titulo do Site. body/body: este e o corpo do nosso documento e nele que ficara contido todo o conteúdo referentes ao nosso site no exemplo acima perceba que temos uma tag

de paragrafo e dentro dela definimos um texto básico.

Formatando um Documento HTML5
A formatação de texto em HTML e CSS visa um melhor resultado para a apresentação das suas páginas. Primeiro é preciso entender que a estruturação do texto é geito emm HTML e a formatação é feita em CSS, salvo em alguns casos onde algumas formatações podem ser feitas em HTML.
Imagem, Áudio e Vídeo
Para quem ainda não sabe, as tag's audio e video são tag's de media que foram criadas, entre outros motivos, para acabar com a dependência de plugin's para a reprodução de áudio e vídeo nas páginas.
Trabalhando com Vínculos, Links e Microdata
A HTML usa a tag a (âncora) para criar um vínculo (link) com outro documento.
Trabalhando com Listas e Tabelas
Tabelas são elementos utilizados com frequência para exibir dados de forma organizada em linhas e colunas. Listas são elementos usados para organizar e ordenar itens que estão relacionados de alguma forma. No HTML é possível criar três tipos de listas.
